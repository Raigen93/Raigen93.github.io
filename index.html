Codex

The Codex is a unique feature that allows players to access information about other characters they have encountered and their actions. It categorizes characters into two groups: those with nicknames (known) and those without (unknown). Selecting a character opens an interface showcasing various ways to interact with them. At the bottom of the interface, players can view a detailed list of all the actions they have observed that character perform.

Custom In-line Editing

This project involved developing a system that allows users to select different data tables for display. Based on the selected table, a dynamic table and corresponding entry form are generated, reflecting the retrieved data. Users can easily interact with the entries in the table, as each one features seamless inline inputs. These inputs include dropdowns for enum values, as well as fields for text and numbers, enabling users to edit entries directly by modifying the input fields.

Note Viewer

The Note Viewer is a reusable component that allows users to view notes by calling a function and passing in the note's data, which can include both images and text. Upon invocation, the function dynamically attaches a modal to the body of the page, enabling the user to view the note in a dedicated space. If there are multiple notes, pagination is supported, displaying one note at a time. Additionally, calling the function automatically loads all the necessary functions required for the Note Viewer's functionality, ensuring seamless interaction and a smooth user experience.
